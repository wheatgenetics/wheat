Wheat Theme for Poland Lab Websites
===

This is a WordPress theme that is being used for the following Poland Lab websites:
* https://wheatgenetics.k-state.edu/
* https://wgrc-iucrc.k-state.edu/
* https://wgrc.k-state.edu/
